## Lithology Classification & Geophysical Signatures

### 1. COAL (All Ranks)

#### Density Log Responses:
- **Short-space density (g/cc):** 0.7 - 1.8
  - Minimum: 0.7 g/cc (low-rank coal, lignite)
  - Maximum: 1.8 g/cc (high-rank anthracite)
  - Typical: 1.2 - 1.5 g/cc (bituminous coal)
  - Source: USGS Circular 891, ScienceDirect

- **Long-space density (g/cc):** Typically 0.05-0.15 g/cc higher than short-space
  - Difference: 0.05-0.15 g/cc due to bed boundary effects
  - Source: Various technical papers

#### Gamma Ray Responses:
- **Total Gamma Ray (API units):** 15 - 150
  - Minimum: 15 API (clean coal)
  - Maximum: 150 API (uraniferous coal)
  - Typical: 20 - 50 API (most coal seams)
  - Source: AAPG Wiki, USGS Circular 891

- **Spectral Gamma Ray:**
  - Potassium (K): Low to moderate
  - Thorium (Th): Typically low
  - Uranium (U): Variable, can be high in uraniferous coal
  - Source: Technical papers on spectral logging

#### Resistivity Responses:
- **Deep Resistivity (ohm-m):** 100 - 10,000+
  - Minimum: 100 ohm-m (high-moisture lignite)
  - Maximum: 10,000+ ohm-m (dry anthracite)
  - Typical: 500 - 2,000 ohm-m (bituminous coal)
  - Source: USGS Circular 891, ScienceDirect

- **Shallow Resistivity:** Typically 20-50% lower than deep resistivity
  - Difference due to mud invasion and near-wellbore effects
  - Source: Various case studies

### 2. SANDSTONE (Quartz)

#### Density Log Responses:
- **Short-space density (g/cc):** 2.20 - 2.65
  - Minimum: 2.20 g/cc (high porosity sandstone)
  - Maximum: 2.65 g/cc (quartz matrix density)
  - Typical: 2.35 - 2.50 g/cc
  - Source: AAPG Wiki, USGS Circular 891

- **Long-space density:** Similar to short-space in homogeneous formations

#### Gamma Ray Responses:
- **Total Gamma Ray (API units):** 15 - 30 (rarely to 200)
  - Minimum: 15 API (clean quartz sandstone)
  - Maximum: 200 API (arkosic sandstone with K-feldspar)
  - Typical: 20 - 25 API
  - Source: AAPG Wiki

#### Resistivity Responses:
- **Deep Resistivity (ohm-m):** 10 - 1,000+
  - Minimum: 10 ohm-m (shaly, water-saturated)
  - Maximum: 1,000+ ohm-m (clean, hydrocarbon-saturated)
  - Typical: 50 - 200 ohm-m
  - Source: Various technical papers

### 3. SHALE

#### Density Log Responses:
- **Short-space density (g/cc):** 2.20 - 2.65+
  - Minimum: 2.20 g/cc (undercompacted, overpressured)
  - Maximum: 2.65+ g/cc (highly compacted)
  - Typical: 2.40 - 2.55 g/cc
  - Increases with depth and age
  - Source: AAPG Wiki, USGS Circular 891

#### Gamma Ray Responses:
- **Total Gamma Ray (API units):** 60 - 150
  - Minimum: 60 API (sandy shale)
  - Maximum: 150 API (organic-rich shale)
  - Typical: 80 - 120 API
  - Source: AAPG Wiki

- **Organic-rich shale:** 100 - 250 API
  - High uranium content associated with organic matter

#### Resistivity Responses:
- **Deep Resistivity (ohm-m):** 1 - 50
  - Minimum: 1 ohm-m (conductive shales)
  - Maximum: 50 ohm-m (silty shales)
  - Typical: 5 - 20 ohm-m
  - Source: Various technical papers

### 4. LIMESTONE

#### Density Log Responses:
- **Short-space density (g/cc):** 2.60 - 2.80
  - Minimum: 2.60 g/cc (porous limestone)
  - Maximum: 2.80 g/cc (dense limestone)
  - Typical: 2.71 g/cc (calcite matrix)
  - Source: AAPG Wiki

#### Gamma Ray Responses:
- **Total Gamma Ray (API units):** 10 - 40
  - Minimum: 10 API (clean limestone)
  - Maximum: 40 API (argillaceous limestone)
  - Typical: 15 - 25 API
  - Source: AAPG Wiki

#### Resistivity Responses:
- **Deep Resistivity (ohm-m):** 50 - 10,000+
  - Highly variable depending on porosity and fluid content
  - Can be mistaken for coal on resistivity logs alone
  - Source: USGS Circular 891

### 5. DOLOMITE

#### Density Log Responses:
- **Short-space density (g/cc):** 2.80 - 2.90
  - Minimum: 2.80 g/cc (porous dolomite)
  - Maximum: 2.90 g/cc (dense dolomite)
  - Typical: 2.85 g/cc
  - Source: AAPG Wiki

#### Gamma Ray Responses:
- **Total Gamma Ray (API units):** 15 - 40 (rarely to 200)
  - Minimum: 15 API (clean dolomite)
  - Maximum: 200 API ("hot" dolomite with uranium)
  - Typical: 20 - 30 API
  - Source: AAPG Wiki

### 6. ANHYDRITE

#### Density Log Responses:
- **Short-space density (g/cc):** 2.90 - 3.00
  - Consistent, straight-line density log
  - Source: AAPG Wiki

#### Gamma Ray Responses:
- **Total Gamma Ray (API units):** 8 - 15
  - Very low natural radioactivity
  - Source: AAPG Wiki

### 7. HALITE (Salt)

#### Density Log Responses:
- **Short-space density (g/cc):** 2.10 - 2.20
  - Source: AAPG Wiki

#### Gamma Ray Responses:
- **Total Gamma Ray (API units):** 8 - 15
  - Very low natural radioactivity
  - Source: AAPG Wiki

## Basin-Specific Case Studies

### Bowen Basin (Australia)
- **Coal density range:** 1.3 - 1.7 g/cc
- **Gamma ray range:** 25 - 60 API
- **Resistivity range:** 300 - 1,500 ohm-m
- **Data source:** Automated classification study of 1230 samples from 263 boreholes
- **Special characteristics:** Metamorphosed coal near igneous intrusions shows higher density

### Illinois Basin (USA)
- **Coal density range:** 1.2 - 1.6 g/cc
- **Gamma ray range:** 20 - 80 API
- **Resistivity range:** 200 - 1,000 ohm-m
- **Data source:** USGS studies on coal quality characterization
- **Special characteristics:** Computer-processed log data successfully determined moisture and ash content

### Ruhr Basin (Germany)
- **Coal density range:** 1.4 - 1.8 g/cc
- **Gamma ray range:** 30 - 70 API
- **Resistivity range:** 400 - 2,000 ohm-m
- **Data source:** European coal mining studies
- **Special characteristics:** Well-documented in European geological literature

### Powder River Basin (USA)
- **Coal density range:** 0.9 - 1.4 g/cc (low-rank subbituminous)
- **Gamma ray range:** 15 - 40 API
- **Resistivity range:** 100 - 500 ohm-m
- **Special characteristics:** Low-density, high-moisture coal

### Appalachian Basin (USA)
- **Coal density range:** 1.3 - 1.7 g/cc
- **Gamma ray range:** 25 - 100 API
- **Resistivity range:** 500 - 3,000 ohm-m
- **Special characteristics:** High-rank bituminous coal

## Conflicting/Varied Data Analysis

### Major Conflicts Identified:

1. **Coal Gamma Ray Values:**
   - Some sources: 15-50 API (typical clean coal)
   - Other sources: Up to 150 API (uraniferous coal)
   - Resolution: Uranium content varies regionally

2. **Sandstone Resistivity:**
   - Clean sandstone: 100-1,000+ ohm-m
   - Shaly sandstone: 10-100 ohm-m
   - Resolution: Clay content significantly affects resistivity

3. **Dolomite Gamma Ray:**
   - Typical: 15-40 API
   - "Hot" dolomite: Up to 200 API
   - Resolution: Uranium enrichment in specific formations

4. **Coal Density vs Rank:**
   - Low-rank coal: 0.7-1.2 g/cc
   - High-rank coal: 1.5-1.8 g/cc
   - Resolution: Rank progression increases density

## Technical Papers Summary

### Key Papers on Coal Geophysical Signatures:

1. **Wood et al. (1983)** - USGS Circular 891
   - Comprehensive guide to coal bed identification from logs
   - Density range: 0.7-1.8 g/cc for coal
   - Emphasizes need for multiple log types

2. **Bond et al. (1971)** - Illinois Basin Study
   - Computer-processed log data for coal composition
   - Successful determination of moisture and ash content

3. **Various Bowen Basin Studies (2019)**
   - Machine learning classification of metamorphosed coal
   - 1230 samples from 263 boreholes
   - Density variations near igneous intrusions

4. **Gamma and Resistivity Logs for Gondwana Coal (2014)**
   - Bangladesh coal basin study
   - High resistivity, low gamma in coal seams
   - Quantitative relationships established

## Recommendations for CoalLog v3.1 Implementation

1. **Use Multi-Log Approach:** Never rely on single log type for lithology identification
2. **Regional Calibration:** Adjust expected ranges based on local geology
3. **Rank Considerations:** Account for coal rank in density and resistivity values
4. **Spectral Gamma Ray:** Use when available for better clay/organic matter discrimination
5. **Quality Control:** Reject ambiguous logs to avoid erroneous data
