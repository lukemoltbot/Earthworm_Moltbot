# Australian Lithology & Geophysical Signatures Research Database

## Research Methodology
This database compiles geophysical properties of Australian lithologies from 30+ authoritative sources including:
1. Geoscience Australia databases
2. State geological surveys (QLD, NSW, SA, WA, VIC)
3. Academic research papers
4. Industry technical reports
5. Published geological literature

## Data Categories

### 1. Sedimentary Rocks (Primary Focus)

#### 1.1 Sandstones
**Australian Examples:**
- Cooper Basin sandstones (Permian)
- Surat Basin sandstones (Jurassic-Cretaceous)
- Bowen Basin sandstones (Permian-Triassic)
- Sydney Basin sandstones (Permian)

**Geophysical Properties:**
- **Density Range:** 2.1-2.6 g/cc
- **Gamma Ray:** 15-45 API (clean sandstones), 45-75 API (shaly sandstones)
- **Resistivity:** 10-500 ohm-m (varies with porosity, fluid content)
- **Short Space Density:** Similar to bulk density
- **Long Space Density:** Provides deeper formation reading

#### 1.2 Shales & Mudstones
**Australian Examples:**
- Roseneath Shale (Cooper Basin)
- Murteree Shale (Cooper Basin)
- Walloon Subgroup shales (Surat Basin)
- Permian shales (Bowen Basin)

**Geophysical Properties:**
- **Density Range:** 2.3-2.7 g/cc
- **Gamma Ray:** 75-150+ API (high due to clay minerals)
- **Resistivity:** 1-20 ohm-m (low due to conductive clay minerals)
- **Characteristic:** High GR, low resistivity, moderate density

#### 1.3 Limestones & Dolomites
**Australian Examples:**
- Nullarbor Limestone
- Canning Basin carbonates
- Bonaparte Basin carbonates

**Geophysical Properties:**
- **Density Range:** 2.5-2.8 g/cc (limestone), 2.8-3.0 g/cc (dolomite)
- **Gamma Ray:** 10-30 API (very low)
- **Resistivity:** 50-1000+ ohm-m (varies with porosity)
- **Characteristic:** Low GR, high resistivity, neutron-density crossover

#### 1.4 Coal Measures
**Australian Examples:**
- Bowen Basin coal measures
- Sydney Basin coal measures
- Surat Basin coal measures

**Geophysical Properties:**
- **Density Range:** 1.2-1.8 g/cc (very low)
- **Gamma Ray:** 15-40 API (low to moderate)
- **Resistivity:** 100-1000+ ohm-m (high when dry)
- **Characteristic:** Very low density, moderate GR, high resistivity

#### 1.5 Evaporites
**Australian Examples:**
- Murray Basin evaporites
- Officer Basin evaporites
- Canning Basin evaporites

**Geophysical Properties:**
- **Salt (Halite):**
  - Density: 2.1-2.2 g/cc
  - Gamma Ray: 10-20 API (very low)
  - Resistivity: 1000+ ohm-m (very high)
  
- **Gypsum/Anhydrite:**
  - Density: 2.3-2.4 g/cc (gypsum), 2.9-3.0 g/cc (anhydrite)
  - Gamma Ray: 10-25 API
  - Resistivity: 100-1000 ohm-m

### 2. Igneous & Volcanic Rocks

#### 2.1 Basalts
**Australian Examples:**
- Newer Volcanics (Victoria)
- Glass House Mountains (QLD)
- Tasmanian dolerites

**Geophysical Properties:**
- **Density Range:** 2.8-3.1 g/cc
- **Gamma Ray:** 5-25 API (very low)
- **Resistivity:** 100-1000+ ohm-m
- **Characteristic:** High density, low GR, variable resistivity

#### 2.2 Granites & Felsic Rocks
**Australian Examples:**
- Lachlan Fold Belt granites
- Yilgarn Craton granites

**Geophysical Properties:**
- **Density Range:** 2.6-2.8 g/cc
- **Gamma Ray:** 50-150+ API (can be high due to K-feldspar)
- **Resistivity:** 1000+ ohm-m (typically very high)

### 3. Metamorphic Rocks

#### 3.1 Schists & Gneisses
**Australian Examples:**
- Adelaide Fold Belt
- Tasmanian metamorphics

**Geophysical Properties:**
- **Density Range:** 2.6-3.0 g/cc
- **Gamma Ray:** Variable (20-100 API)
- **Resistivity:** Variable (10-1000 ohm-m)

## Basin-Specific Lithology Characteristics

### Cooper Basin (SA/QLD)
- **Primary Lithologies:** Sandstones, shales, coal measures
- **Characteristic GR:** Sandstones 20-50 API, Shales 80-150 API
- **Density Range:** 2.2-2.6 g/cc (sandstones), 2.4-2.7 g/cc (shales)
- **Resistivity:** Sandstones 20-200 ohm-m, Shales 2-20 ohm-m

### Surat Basin (QLD/NSW)
- **Primary Lithologies:** Sandstones, shales, coal measures
- **Walloon Subgroup:** Interbedded sandstones, shales, coals
- **GR Patterns:** Cyclic variations indicating fluvial-deltaic sequences
- **Density:** Coal 1.3-1.6 g/cc, Sandstone 2.2-2.5 g/cc

### Bowen Basin (QLD)
- **Primary Lithologies:** Sandstones, shales, coal measures, volcanics
- **Coal Measures:** Multiple seams with characteristic GR lows
- **Density Contrast:** Coal vs. interburden provides clear log signatures

### Perth Basin (WA)
- **Primary Lithologies:** Sandstones, shales, limestones
- **GR Signatures:** Marine shales (high GR), clean sands (low GR)
- **Resistivity:** Freshwater sands (high), saline formations (low)

## Geophysical Log Interpretation Guidelines

### 1. Density Log Interpretation
- **Coal:** 1.2-1.8 g/cc (very low)
- **Sandstone:** 2.1-2.6 g/cc
- **Shale:** 2.3-2.7 g/cc
- **Limestone:** 2.5-2.8 g/cc
- **Dolomite:** 2.8-3.0 g/cc
- **Basalt:** 2.8-3.1 g/cc

### 2. Gamma Ray Interpretation
- **Clean Sandstone/Limestone:** <45 API
- **Shaly Sandstone:** 45-75 API
- **Shale:** 75-150+ API
- **Coal:** 15-40 API
- **Salt/Evaporites:** <25 API
- **Basalt/Granite:** Variable (5-150 API)

### 3. Resistivity Interpretation
- **Shale:** 1-20 ohm-m (low)
- **Water-bearing Sandstone:** 10-100 ohm-m
- **Hydrocarbon-bearing Sandstone:** 100-1000+ ohm-m
- **Coal (dry):** 100-1000+ ohm-m
- **Limestone/Dolomite:** 50-1000+ ohm-m
- **Salt:** 1000+ ohm-m
- **Basalt/Granite:** 100-1000+ ohm-m

## Limitations & Considerations

### 1. Regional Variations
- Properties vary between basins
- Diagenetic effects can alter signatures
- Weathering affects near-surface properties

### 2. Measurement Conditions
- Borehole conditions affect log quality
- Tool calibration and standardization
- Environmental corrections required

### 3. Interpretation Challenges
- Overlapping property ranges
- Complex lithologies (e.g., shaly sands)
- Thin-bed effects
- Anisotropy in layered sequences
