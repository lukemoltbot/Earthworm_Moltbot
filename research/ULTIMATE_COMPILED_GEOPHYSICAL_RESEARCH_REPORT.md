## PART 1: COALLOG V3.1 STANDARD RESEARCH

### 1.1 Core Lithology Types & Codes

| CoalLog Code | Lithology Type | Description | Typical Environment |
|--------------|----------------|-------------|---------------------|
| **COAL** | Coal | Carbonaceous rock >50% carbonaceous material | Swamp, delta plain |
| **SS** | Sandstone | >50% sand-sized grains (0.0625-2mm) | Fluvial, deltaic, beach |
| **SILT** | Siltstone | >50% silt-sized grains (0.0039-0.0625mm) | Prodelta, floodplain |
| **MUD** | Mudstone | Fine-grained clastic, non-fissile | Low-energy marine |
| **CLAY** | Claystone | >50% clay minerals | Low-energy marine |
| **SH** | Shale | Fissile fine-grained clastic | Deep marine, prodelta |
| **LS** | Limestone | >50% calcite | Marine platform, reef |
| **DOL** | Dolomite | >50% dolomite | Sabkha, diagenetic |

### 1.2 Characteristic Geophysical Signatures

#### **Density (g/cm³)**
- **Coal:** 0.7-1.8 (very low, rank-dependent)
- **Sandstone:** 2.2-2.65 (porosity-dependent)
- **Shale:** 2.4-2.8 (compaction-dependent)
- **Limestone:** 2.71 (pure calcite)
- **Dolomite:** 2.87 (pure dolomite)

#### **Gamma Ray (API)**
- **Coal:** 15-150 (typically low <30, but uraniferous up to 150+)
- **Clean Sandstone:** 15-30 (low radioactivity)
- **Shale:** 60-150 (very high, clay/organic content)
- **Limestone:** 10-40 (low to moderate)
- **Dolomite:** 15-40 (low to moderate)

#### **Resistivity (Ω·m)**
- **Coal:** 10-10,000+ (very high, moisture-dependent)
- **Sandstone:** 1-10,000+ (fluid-dependent)
- **Shale:** 1-50 (very low, conductive clay minerals)
- **Limestone:** 10-10,000+ (variable, porosity-dependent)
- **Dolomite:** 10-10,000+ (similar to limestone)

#### **Neutron Porosity (%)**
- **Coal:** 50-60+ p.u. (high due to carbon content)
- **Sandstone:** 15-30% (true porosity)
- **Shale:** 30-60% (exaggerated, clay-bound water)
- **Limestone:** 0-30% (true porosity when clean)

### 1.3 Diagnostic Crossplot Patterns

#### **Neutron-Density Crossplot**
- **Coal:** Both very high (neutron and density high apparent porosity)
- **Sandstone (water):** Density left of neutron by ~2 divisions
- **Sandstone (gas):** Increased separation (~5 divisions)
- **Limestone:** No separation (overlay exactly)
- **Dolomite:** Density right of neutron by 4-6 divisions
- **Shale:** Neutron left of density (large separation)

#### **Gamma Ray vs Resistivity**
- **Coal:** Low GR + High Resistivity
- **Clean Sandstone:** Low GR + High Resistivity
- **Shaly Sandstone:** Moderate GR + Moderate Resistivity
- **Shale:** High GR + Low Resistivity
- **Limestone:** Low GR + Variable Resistivity

## PART 3: LAS CURVE ANALYSIS & IMPLEMENTATION FINDINGS

### 3.1 Critical Issues Identified

#### **1. Curve Scaling Problems**
- **Root Cause:** Depth scale mismatch and viewport synchronization issues
- **Bug:** `scroll_to_depth` method assumes valid `current_height`, breaks if view not initialized
- **Fix:** Add validation for `current_height`, ensure view range initialization

#### **2. Gamma Ray Alignment Issues**
- **Root Cause:** Dual-axis synchronization bug - only X-axis linked, missing Y-axis link
- **Critical Bug:** `self.gamma_viewbox.setXLink(self.plot_item)` only links X-axis
- **Missing:** `self.gamma_viewbox.setYLink(self.plot_item.vb)` for Y-axis synchronization
- **Fix:** Add Y-axis linking, connect scrolling signals to update gamma viewbox

#### **3. Range Normalization Strategies**
- **Current:** Gamma Ray 0-300 API vs Density 0-4 g/cc (75:1 ratio)
- **Options:**
  - A: Dual-Axis System (current, needs fixing)
  - B: Normalized Ranges (0-1)
  - C: Smart Auto-ranging
  - D: Logarithmic Scaling
- **Recommendation:** Fix dual-axis bugs, add normalization toggle

#### **4. Visibility Toggle Architecture**
- **Current:** `set_curve_visibility()` method exists with curve name mapping
- **Issues:** UI controls not connected, no state persistence
- **Fix:** Connect UI controls, add state save/restore mechanism
